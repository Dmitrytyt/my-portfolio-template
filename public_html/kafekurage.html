Сайт кафе Кураж

Небольшому кафе в Анапе потребовался сайт для доставки роллов, пиццы, суши.
В результате работы над данным сайтом было сделано:
  • сверстан html шаблон на основе bootstrap3
  • шаблон был интегрирован с программной частью сайта
  • написана авторизация, регистрация, напоминание забытого пароля
  • написана авторизация через популярные социальные сети
  • написана корзина для добавления заказов
  • разработан личный кабинет для пользователей
  • в личном кабинете пользователя реализована возможность просмотра заказов
  • для администратора сайта написана панель управления контентом, заказами
  • запрограммированы формы для отправки данных (заявки, отзывы, контакты)
  • сайт был наполнен контентом (более 200 блюд)
  • произведена внутренняя оптимизация под поисковые системы
  • а также другой функционал.
