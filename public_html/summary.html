Резюме

Скачать резюме в формате .pdf

Приветствую тебя потенциальный работодатель! В этом небольшом, но содержательном резюме, я бы хотел рассказать о себе как о веб-разработчике.

В настоящий момент я удалённо выполняю заказы по разработке и сопровождению сайтов.

Имеется опыт как FrontEnd, так и BackEnd разработки. Знаком с seo-продвижением.

При разработке использую следующие технологии: PHP • JavaScript • ООП • MySQL • Yii framework • Codeigniter • Git • Ajax • Sass • Twitter Bootstrap • Gulp • HTML5 • Jade

Увидеть примеры моих работ, а также получить дополнительную информацию можно на данном сайте.

Мои интересы:
Я заинтересован в долгосрочном сотрудничестве в проектах с различным уровнем сложности. Пока я рассматриваю предложения только с удалённой работой, но в случае необходимости готов к переезду. Я также заинтересован в развитии своих профессиональных качеств и навыков.

Предыдущее место работы - это небольшая компания, имеющая 2 сайта с небольшой посещаемостью от 2000 - 15000 человек в сутки. В этой компании с 2010 по 2017 год я занимался сопровождением и доработкой существующего функционала сайтов компании, а также небольшого числа сайтов визиток наших клиентов.

Также во время работы над этими проектами, мною был разработан еще один новый сайт для компании схожей тематики, который работает и по сей день, принося отличный доход!

За время работы над сайтами было сделано:

  • были свёрстаны новые макеты для сайтов
  • написан личный кабинет для пользователей
  • написан личный кабинет для менеджеров (модераторов)
  • разработана небольшая CRM-система, с возможностью напоминаний о совершении каких-либо действий, учёт платежей за размещение информации
  • смс-рассылка о забытых паролях, смс-оповещение о поступивших заявках клиентам через собственные шлюзы смс-рассылок
  • резервное копирование данных

В проектах использовались следующие технологии:

PHP 5.4, MySQL, Bootstrap 3, JavaScript, jQuery, Less, Git
